About This App

This is a client-side To-Do List application. All data is stored in your browser's localStorage, so no server is required.


Features:

  • ✔ Add new tasks with priority levels
  • ✔ Edit existing tasks
  • ✔ Delete tasks
  • ✔ Mark tasks as completed
  • ✔ Filter tasks by status
  • ✔ Data persists across browser sessions

Technologies Used:

  1. HTML5 — Structure & Semantics
  2. CSS3 — Styling & Layout
  3. JavaScript (ES6) — Interactivity & Logic
